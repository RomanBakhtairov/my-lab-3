При запуске появляется поле с шариком посередине. Нажимая на кнопку, появляется "гравитация". 
При столкновении шарика с полом с некоторой вероятностью может появится другой шарик.
Чёрным шариком можно управлять стрелочками: влево, вправо, вниз. 
Замечание! управление работает как воздействие на шарик силой, так что к нему придется привыкать
